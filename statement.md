## Problem Statement
Understanding movie reviews manually is time-consuming.  
There is a need for a simple tool that can take a text-based movie review and classify it as **positive** or **negative** using a basic algorithm.

---

## Scope of the Project

This project focuses on:
- Sentiment analysis using word frequency counting  
- Random dataset generation  
- Basic string processing  
- Simple rule-based prediction  
- Command-line interaction  
- Beginner-friendly demonstration of data processing concepts  

The project **does not** include advanced NLP or machine learning models.

---

## Target Users

- Students learning Python  
- Beginners exploring text processing  
- Anyone who wants a simple classification program  
- Faculty evaluating VITyarthi projects  

---

## High-Level Features

1. **Random Dataset Generation**  
   Automatically generates positive and negative reviews for training.

2. **Word Frequency Training**  
   Counts how often each word appears in positive vs negative reviews.

3. **Sentiment Prediction**  
   Predicts the sentiment of a user-entered movie review.

4. **Continuous Interaction**  
   Accepts multiple reviews until the user types `exit`.

5. **Error Handling**  
   Warns the user when blank input is entered.
